# DAP UR HOMIES - Cooperative Multiplayer Moves for Minecraft

A Fabric mod for Minecraft 1.21.1 and 1.21.11 that adds cooperative player-to-player interaction mechanics. Dap your friends, high-five, hug, grab & throw, and pull off combo chains with perfectly timed QTEs.

## Features

- Charged Dap System — Hold G to charge, release near a partner. Timing and speed determine the tier (Weak → Great → Legendary → Fire → Heaven)
- Dap QTE
- High Five — Press H to raise your hand. Connect with another player for speed-based impact effects + combo follow-ups
- Grab & Throw — Pick up other players and yeet them
- Human Shield — Use a grabbed player as a shield
- Mario Jump — Bounce off another player's head
- Hug System
- Fire Dap Combo
- Heaven Dap

### Requirements for 1.21.1:

- Fabric Loader 0.18.4+
- Fabric API
- Player Animation Library (PAL) — for all custom animations  https://modrinth.com/mod/player-animation-library

### Requirements for 1.21.11:

- Fabric Loader 0.18.4+
- Fabric API
- Player Animation Library (PAL) — for all custom animations  https://modrinth.com/mod/player-animation-library
